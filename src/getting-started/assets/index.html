Welcome to Visual Studio Code!
In this 1-minute tutorial, we'll show you how to create a quick-start Java program in VS Code.

Setup the Workspace

To setup the workspace, simply open a folder by:

Windows
File > Open Folder...
Mac
File > Open...

No Import Needed

VS Code Java works directly with folders that have source code. If the folder you open has project/build configuration files like pom.xml or build.gradle, they will be picked up automatically and the folder will be treated as a project.

Create a Class

A program needs an entry and a Java program needs a class to host its entry. To create a class for our quick-start program, create a file and set its name to QuickStart.java.
Now you can put the code in the new file: 

class QuickStart {
    public static void main(String[] args) {
        System.out.println("Hello, World.");
    }
}

Run the program

To run the program, press F5. By default, the program is launched in the Intergrated Terminal. You should already see the output there.

How to Debug?

When you press F5, you are already debugging. Try set some breakpoint by clicking on the line numbers before each code line and run the program again. You'll see the execution paused at the breakpoint.
While debugging, switch to the Debug View to see the variables and call stacks.

Congratulations!

Now the quick-start program is running and you are free to build more.
